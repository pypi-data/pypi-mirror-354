MCP Feedback Enhanced

v{{ version }}
一個強大的 MCP 伺服器，為 AI 輔助開發工具提供人在回路的互動回饋功能。支援 Qt GUI 和 Web UI 雙介面，並具備圖片上傳、命令執行、多語言等豐富功能。
📂 GitHub 專案
https://github.com/Minidoracat/mcp-feedback-enhanced
