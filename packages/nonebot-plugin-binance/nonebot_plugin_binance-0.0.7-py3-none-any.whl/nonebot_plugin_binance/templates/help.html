BN小助手 帮助手册

基础指令

bn bind
【私聊】绑定您的API密钥
【请在您绝对信任机器人提供方的情况下绑定！！！】
bn bind [key] [secret]
bn unbind
解绑您的API密钥
bn unbind
bn status
查询绑定状态和API密钥权限
bn status

行情查询

bn [货币名]
【快捷】查该货币对USDT价格
bn btc
bn k [货币名]
【快捷】查该货币1日K线
bn k btc
bn price / bn p
获取指定交易对最新价格
bn p btc-usdt
bn kline / bn k
获取K线图 (含快捷指令)
bn k eth-usdt 1h
K线周期单位: m=分钟, h=小时, d=天, w=周, M=月. (例: 1m, 1h, 1d)

资产与交易

bn balance / bn bal
查询账户资产
bn bal
bn open
查询当前挂单 (可指定交易对)
bn open btc-usdt
bn cancel
撤销指定订单
bn cancel btc-usdt [订单ID]
bn order
下单 (默认关闭)
bn order buy btc-usdt 0.1

价格预警

bn alert add
【群聊】添加价格预警
bn alert add btc-usdt >120000
bn alert list
查看我设置的所有预警
bn alert list
bn alert remove
移除指定ID的价格预警
bn alert remove [预警ID]
