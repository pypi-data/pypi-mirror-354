# Installation

## Requirements

- Python 3.10 or higher
- pip package manager

## Install from PyPI

```bash
pip install agenticraft
```

## Install from Source

```bash
git clone https://github.com/agenticraft/agenticraft.git
cd agenticraft
pip install -e .
```

## Verify Installation

```python
import agenticraft
print(agenticraft.__version__)
```
